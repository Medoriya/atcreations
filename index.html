1. How graphics designing helps in UX/UI Designing ?

When a designer say, he is a graphics designer he is confident with the fundamental of software and designing skills, which helps him to design user interface and small assets like logo, icons, states, color pallet, edit images and backgrounds, which is needed by a UI Designer anyway, so why depend on other designer when you can help yourself, hence graphics designing skills helps in UX/UI Designing

2. What is ux/ui designing ?

UX design refers to User Experience Design, while UI stands for User Interface Design. Both are equally important to build a strong product. A user’s experience of the product (app) is determined by how they interact with it and User Interface focus on look and feel of it.

we are here to nourish your raw brain using design thinking.
